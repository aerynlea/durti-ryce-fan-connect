# Roadmap

This page outlines the current direction for **Durti-Ryce Fan Connect**. It is meant to show where the project is now, what is planned next, and which larger goals are shaping future development.

## Current State

The app already provides a strong mobile-first fan hub with:

- a branded home experience
- live show discovery
- music and video access
- featured artist profiles
- merch browsing with cart behavior
- links to fan club, booking, and contact paths

## Near-Term Priorities

These are the highest-value next steps for the current version of the app:

- connect newsletter signup to a real email platform
- improve app copy, polish, and consistency
- expand live event and media content
- improve navigation and screen flow
- strengthen merch data handling and cart behavior

## Merch Roadmap

The merch experience is one of the biggest long-term opportunities for the app.

Planned merch milestones include:

- move product data into Supabase
- support a shared cart direction across app and web
- create Stripe checkout sessions from app-managed cart data
- process successful payments with webhook handling
- connect paid orders to Printful fulfillment

## Content Roadmap

Future content improvements may include:

- more artist profiles
- richer show details and announcements
- featured releases and campaign pages
- fan club-exclusive content
- stronger storytelling across home and artist sections

## Product Experience Roadmap

Planned user-experience improvements include:

- cleaner navigation between major app areas
- more native-feeling tab and section transitions
- better status messaging for merch and backend-connected features
- stronger onboarding for first-time users
- clearer calls to action for shows, merch, and fan signup

## Platform And Operations Roadmap

As the project matures, operational work will become more important.

Likely next steps include:

- formal environment setup documentation
- deployment checklists for backend functions
- release testing notes
- production configuration tracking
- better issue and feature planning in GitHub

## Longer-Term Vision

The long-term goal is to make Durti-Ryce Fan Connect feel like a complete digital home for supporters, with a unified fan experience across content, commerce, and community.
