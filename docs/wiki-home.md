# Durti-Ryce Fan Connect Wiki

Welcome to the official project wiki for **Durti-Ryce Fan Connect**.

This app is a mobile-first fan experience built with Expo and React Native. It brings together the core parts of the Durti-Ryce brand in one place so supporters can quickly find shows, explore music, shop merch, discover artist profiles, and stay connected.

## Project Overview

Durti-Ryce Fan Connect is designed to serve as a central hub for fans by combining:

- upcoming shows and ticket links
- music and video access
- merchandise browsing
- fan club entry points
- newsletter signup
- artist profiles
- social, booking, and contact links

## Current App Sections

The app currently includes these primary sections:

- **Home**: branded hero experience, featured content, merch highlights, and newsletter signup
- **Shows**: upcoming live dates, venue details, ticket links, and map links
- **Bookings**: direct booking-related access points
- **Merch**: product browsing, cart behavior, and checkout handoff
- **Artists**: featured profiles for Deron, Hunter Lane, and Monique Renee
- **More**: fan club, socials, contact, and supporting links

## Tech Stack

- **Expo**
- **React Native**
- **JavaScript**
- **Supabase** for planned/shared merch backend support

## Merch Status

The merch experience currently supports:

- browsing featured products
- adding items to a local in-app cart
- reviewing totals and quantities
- continuing to the live Durti-Ryce store for checkout

The project also includes starter backend planning for a future shared merch system using:

- **Supabase**
- **Stripe**
- **Printful**
