# Band News App

A simple Expo + React Native starter app for a band newsletter experience.

## What it includes

- Updates feed for band announcements
- Shows screen with ticket links
- Merch screen with sample items
- Simple tab switching on one page to keep the first version easy to learn

## How to run

1. Open a terminal in this folder.
2. Install dependencies:

```bash
npm install
```

3. Start the app:

```bash
npm start
```
